# Sistema CEPA · Preview

Prototipo interactivo del Sistema CEPA (Centro de Estudios Psicosociales Aplicados, Universidad de Talca).

## Vista en vivo

Abrir [`index.html`](./index.html) en el navegador o visitar la versión desplegada en GitHub Pages.

## Stack

HTML + CSS + React (via CDN Babel standalone). Sin build step.

## Módulos

Dashboard · Ingresos y pacientes · Licencias médicas · Gestión de fármacos · Controles médicos · Seguimiento EPT · Reintegro laboral · Auditoría · Agendamiento · Reportería.
